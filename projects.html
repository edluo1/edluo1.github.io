Boston University Map

Created in 2015, this map is a new way to get around Boston University. Its function is to display the shapes and sizes of buildings in the university, allowing students to better plan their navigation from building to building.

  • Uses the Google Maps API to display polygons representing buildings
  • Extracts data out of an SQLite database to get building data
